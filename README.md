# AKS Cloud Labs

This repository contains labs and exercises for Azure Kubernetes Service (AKS).

## Getting Started

1. Clone this repository
2. Follow the lab instructions in each directory

## Prerequisites

- Azure CLI
- kubectl
- Helm (optional)

## Labs

- Lab 1: Setting up AKS cluster
- Lab 2: Deploying applications
- Lab 3: Scaling and monitoring
